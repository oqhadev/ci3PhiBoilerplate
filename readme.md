### CLI


#CRUD GENERATOR

**Syntax**

php oqha.php create:crudDT <tableName string> <controllerOrModelName string>  <pathTo string> 

**example**

>php oqha.php create:crudDT sayur sayuran admin
>- Applications
>  - Modules
>    - Admin
>      - controller
>        - sayuran.php
>      - model
>        - sayuran_model.php
>      - view
>        - sayuran
>         - sayuran_list.php 
>         - sayuran_form.php
